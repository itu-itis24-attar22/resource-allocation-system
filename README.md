# Resource Allocation System (Iterative Prototype)

This repository contains an iterative prototype for a university resource allocation system.

The project follows an iterative development approach.

## Repository Structure

- `docs/srs/` : Software Requirements Specification
- `docs/analysis/` : Initial domain analysis and large-scale domain model
- `docs/iterations/` : Iteration reports
- `src/` : C++ prototype implementation

## Project Phases

1. System analysis and SRS
2. Initial domain modeling
3. Iterative implementation

## Iterative Development

### Iteration 1
One-time classroom request evaluated using `AvailabilityRule`.

### Iteration 2
One-time classroom request evaluated using `AvailabilityRule` and `CapacityRule`.
